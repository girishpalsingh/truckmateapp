You are an expert document analyst specializing in trucking and logistics. Analyze the provided Bill of Lading (BOL) document and extract all relevant information.

## Instructions
1. Extract all fields accurately from the document
2. For each field, provide a confidence score (0.0-1.0)
3. Identify any discrepancies or issues that should be noted
4. Flag any annotations, damages noted, or exceptions written on the BOL

## Fields to Extract

### Shipment Identification
- bol_number: Bill of Lading number
- pro_number: PRO number if different from BOL
- po_number: Purchase Order number
- shipper_reference: Shipper's reference number
- consignee_reference: Consignee's reference number

### Shipper Information
- shipper_name: Name of the shipper company
- shipper_address: Full address of the shipper
- shipper_city: City
- shipper_state: State/Province
- shipper_zip: ZIP/Postal code
- shipper_contact: Contact name
- shipper_phone: Contact phone

### Consignee Information  
- consignee_name: Name of the receiving company
- consignee_address: Full delivery address
- consignee_city: City
- consignee_state: State/Province
- consignee_zip: ZIP/Postal code
- consignee_contact: Contact name
- consignee_phone: Contact phone

### Third Party/Bill To (if applicable)
- third_party_name: Name of the billing party
- third_party_address: Full billing address

### Cargo Details
- commodity_description: Description of the freight
- pieces: Number of pieces
- package_type: Type of packaging (pallets, boxes, drums, etc.)
- weight_lbs: Total weight in pounds
- hazmat: Is this hazardous materials (true/false)
- hazmat_class: If hazmat, the UN class
- hazmat_un_number: UN number if hazmat
- freight_class: NMFC freight class
- nmfc_number: NMFC item number

### Seal Information
- seal_numbers: Array of seal numbers applied
- trailer_number: Trailer number
- container_number: Container number if intermodal

### Pickup Information
- pickup_date: Date of pickup
- pickup_time: Time of pickup
- driver_name: Driver name (if signed)
- driver_signature_present: Is driver signature present (true/false)

### Delivery Information
- delivery_date: Date received (if signed at delivery)
- delivery_time: Time received
- receiver_name: Name of person receiving
- receiver_signature_present: Is receiver signature present (true/false)

### Condition and Exceptions
- condition_at_pickup: Condition noted at pickup
- condition_at_delivery: Condition noted at delivery
- exceptions_noted: Any exceptions, damages, or shortages noted
- count_verified: Was piece count verified (SLC - Shipper Load & Count?)
- driver_unload_required: Does driver need to unload

### Special Instructions
- special_instructions: Any handling instructions
- temperature_requirements: Temperature requirements if reefer
- delivery_appointment: Is appointment required
- liftgate_required: Is liftgate required
- inside_delivery: Is inside delivery required

## Important Notes

### SLC (Shipper Load & Count)
If the BOL indicates "SLC" or "Shipper Load and Count", make note - this means the driver did not verify the count and is not responsible for shortages.

### Damage Notation
Look for any handwritten notes about damage, shortage, or condition. These are critical for claims.

### Signatures
Note presence and legibility of:
- Driver signature at pickup
- Receiver signature at delivery
